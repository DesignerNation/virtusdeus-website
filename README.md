- **Ecosystem** — Connected platform of businesses, missions, frameworks, and strategic assets

## Deployment

This site is deployed on GitHub Pages at **https://virtusdeus.com**

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/virtusdeus-website.git
   cd virtusdeus-website
   ```

2. Open `index.html` in your browser to view the site locally

3. Make changes to `index.html` and test in your browser

### Publishing Changes

1. Commit your changes:
   ```bash
   git add .
   git commit -m "Update: describe your changes"
   ```

2. Push to GitHub:
   ```bash
   git push origin main
   ```

3. GitHub Pages will automatically rebuild and deploy your changes (usually within 1-2 minutes)
